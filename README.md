# ansible-role-mongo

Single node MongoDB with replica set.

## Requirements

Docker should be installed on your machine.

## Role Variables

- mongo_path: "path/to/your/mongo/folder"

## Dependencies

none

## Example Playbook

```bash
ansible-playbook tests/test.yml
```
